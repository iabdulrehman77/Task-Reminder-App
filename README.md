##  Overview
**Taskly** is an intelligent **AI-powered task reminder application** designed to help users organize their daily activities, stay productive, and never miss important tasks.

By combining traditional task management with **AI-driven assistance**, Taskly helps users plan smarter, prioritize better, and manage time more efficiently ‚Äî all through a clean and user-friendly interface.

---

##  Problem Statement
People often forget tasks, miss deadlines, or feel overwhelmed by unorganized to-do lists.

Taskly addresses this problem by providing:
- Smart task reminders
- AI-assisted task planning
- Simple and distraction-free task management
- Timely notifications and insights
